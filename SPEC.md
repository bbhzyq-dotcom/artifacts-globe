# 如果文物都回家 - 项目规格文档

## 1. 项目概述

**项目名称**: 如果文物都回家 (If Artifacts Could Return Home)

**项目类型**: 数据可视化 / 交互式 3D 地球展示

**核心功能**: 展示大英博物馆 4948 件流失文物在全球的分布，通过时间轴动画模拟文物"从伦敦飞回原产国"的归位过程。

**目标用户**: 对文物归还议题感兴趣的用户、教育工作者、研究者

## 2. 技术栈

- **框架**: Vue 3 + Vite
- **3D 地球**: GlobeStream3D (基于 Three.js)
- **样式**: CSS3 / CSS Variables
- **数据格式**: JSON

## 3. 数据层规格

### 3.1 数据源
- 大英博物馆公开馆藏数据（模拟数据）
- 总量: 4948 件文物
- 覆盖: 99 个国家

### 3.2 数据字段
```json
{
  "id": "文物唯一标识",
  "name": "文物名称",
  "nameEn": "英文名称",
  "country": "原产国",
  "countryCode": "ISO 国家代码",
  "latitude": "纬度",
  "longitude": "经度",
  "acquisitionDate": "入藏时间 (YYYY-MM-DD)",
  "acquisitionYear": "入藏年份",
  "category": "文物类别",
  "description": "描述",
  "image": "图片路径 (本地)"
}
```

### 3.3 数据处理
- 地理坐标映射 (国家 → 经纬度)
- 按入藏时间排序
- 按国家分组统计

## 5. 功能规格

### 5.1 核心功能

**F1: 3D 地球可视化**
- 使用 GlobeStream3D 渲染 3D 地球
- 文物以散点形式分布在原产国位置
- 支持鼠标拖拽旋转、滚轮缩放
- 点击文物散点显示详情

**F2: 时间轴动画**
- 底部时间轴显示入藏年份范围 (约 1750-2025)
- 播放按钮触发动画
- 动画按时间顺序逐件展示文物"飞回"原产国的过程
- 飞行动画: 从伦敦 (51.5, -0.1) 飞往原产国
- 显示当前播放进度 (已飞回 X 件 / 总计 Y 件)

**F3: 国家筛选**
- 左侧面板列出所有 99 个国家
- 显示每个国家的文物数量
- 点击国家:
  - 地球聚焦到该国家区域
  - 高亮该国家的文物
  - 右侧显示该国家文物列表
- 支持搜索特定国家

**F4: 多视图切换**
- **Flat 模式**: 2D 平面地图视角
- **Standing 模式**: 3D 立体视角，相机位于赤道位置
- **Particles 模式**: 文物以粒子形式飘浮在空中
- **Animation 模式**: 时间轴动画模式

### 5.2 交互细节

| 交互 | 行为 |
|------|------|
| 鼠标拖拽地球 | 旋转视角 |
| 滚轮缩放 | 放大/缩小 |
| 点击散点 | 显示文物详情卡片 |
| 点击国家 | 筛选该国文物 |
| 拖动时间轴 | 快速定位年份 |
| 点击播放 | 开始/暂停动画 |

- 当前动画进度: X / 4948
